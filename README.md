# 🍱 Smart Food Redistribution

A modern web-based platform designed to reduce food waste by efficiently connecting food donors with NGOs through a structured and transparent redistribution system.

🔗 **Live Website:**  
https://subhamsje.github.io/Smart-Food-Redistribution/

---

## 📌 Problem Statement

Food waste remains one of the largest global challenges. While surplus food is discarded daily from restaurants, events, and households, millions continue to face hunger.

The core issue is not food scarcity — it is distribution inefficiency.

Smart Food Redistribution aims to bridge this gap using a simple, scalable digital solution.

---

## 🚀 Solution Overview

Smart Food Redistribution provides a streamlined interface that enables:

- 👨‍🍳 Donors to publish surplus food listings  
- 🏥 NGOs to browse available food in real time  
- 📦 NGOs to request food pickups  
- 📊 Automatic tracking of food saved and estimated people fed  

The platform emphasizes clarity, usability, and measurable social impact.

---

## ✨ Key Features

- 📊 Interactive dashboard with live statistics  
- 📝 Donor portal with structured food submission form  
- 🏥 NGO feed with pickup request functionality  
- 📈 Impact visualization (Food saved & People fed)  
- 🎨 Modern glassmorphism UI design  
- 📱 Fully responsive layout  
- ⚡ Smooth transitions and animations
